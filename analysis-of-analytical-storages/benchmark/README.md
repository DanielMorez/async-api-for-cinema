### Storage benchmark

1. MongoDB. 
- массив данных состоит из 148000 документов
- Выборка одного документа происходит на MacBook M1 - за 166 ms, на Ubuntu (Ryzen 5 5600H) по результатам 3-х испытаний - за 0,9 ms, 1,15 ms, 1,02 ms;
